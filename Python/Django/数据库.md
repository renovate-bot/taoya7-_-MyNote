### Django配置连接数据库

`settings.py`

```python
DATABASES = {
    'default': {
        #数据库引擎（splite3/mysql/oracle）
        'ENGINE': 'django.db.backends.mysql',
        # 'NAME': os.path.join(BASE_DIR, 'db.sqlite3'),
        #数据库名字
        'NAME':'python_test',
        #数据库用户名
        'USER':'root',
        #数据库密码
        'PASSWORD':'',
        #数据库主机地址
        'HOST':'127.0.0.1',
        #数据库端口号
        'PORT':'3306'
    }
}
```
