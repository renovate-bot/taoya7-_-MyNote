##  快捷键

| 快捷键 | 说明                               |
| ------ | ---------------------------------- |
| F2     | 下断点                             |
| F3     | 加载一个可执行程序                 |
| F4     | 运行到选定位置                     |
| F7     | 单步步入到下一条命令。遇到Call进入 |
| F8     | 单步步过到下一条命令。遇到Call跳过 |
| F9     | 让程序继续执行                     |
| F12    | 暂停                               |

| 说明     | 快捷键   |
| -------- | -------- |
| 跳转地址 | `Ctrl+G` |
|          |          |
|          |          |
