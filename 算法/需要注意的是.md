# 必须要背的代码

> 全排列

```java
/*全排列*/
public void Af(int[] arr, int k){
    if(k == arr.length-1){
        System.out.println(Arrays.toString(arr));
        return;
    }
    for(int i=k; i<arr.length; i++){
        int tmp = arr[k];
        arr[k] = arr[i];
        arr[i] = tmp;
        Af(arr,k+1);
        tmp = arr[k];
        arr[k] = arr[i];
        arr[i] = tmp;
    }
}
```
