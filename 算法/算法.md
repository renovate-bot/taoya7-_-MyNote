```
public class Main {
	public static void main(String[] args) {
		Scanner in = new Scanner(System.in);
		String line = in.nextLine();
		//数据结构
		Stack<Integer> stack = new Stack<Integer>();
		TreeMap<Integer, Integer> res = new TreeMap();

		/*开始解析每个字符*/
		for (int i = 0; i < line.length(); i++) {
			char c = line.charAt(i);
			if (c == '(') {
				stack.push(i + 1);//因为计数从1开始
			} else {
				Integer pop = stack.pop();
				res.put(pop, i + 1);
			}
		}

		System.out.println(res);
	}
}
```

### 最大数

```java
public class Main {
    public static void main(String[] args){
        int[] arr = {6,10,32,8,19,20,2,14};
        int max = findMax(arr);
        System.out.println(max);
    }
    public static int findMax(int[] arr){
        int max = Max(arr,0,arr.length-1);
        return max;
    }

    public static int Max(int[] arr, int low, int high){
        if(low==high){
            return arr[low];
        }else{
            int mid = (low+high)/2;
            int leftNum = Max(arr,low,mid);
            int rightNum = Max(arr,mid+1,high);
            return Math.max(leftNum,rightNum);
        }
    }

}
```

完全数等于其所有因子的和。6=1+2+3

- 完全数都是偶数
- 完全数都可以表示连续的自然数的和

6=1+2+3
28=1+2+3+4+5+6+7

```
public class Main {
	public static boolean f(int n) {

		int sum = 0; //求和
		for (int i = 1; i < n; i++)//求因子
			if (n % 2 == 0 && n % i == 0)
				sum += i;

		if (n == sum)//比较
			return true;
		return false;
```

### 质数

厄拉多塞筛法 介绍：改算法在寻找素数时，采用了一种与众不同的方法：先将 2－N 的各数放入表中，然后在 2 的上面画一个圆圈，然后划去 2 的其他倍数；第一个既未画圈又没有被划去的数是 3，将它画圈，再划去 3 的其他倍数；现在既未画圈又没有被划去的第一个数是 5，将它画圈，并划去5的其他倍数……依次类推，一直到所有小于或等于N的各数都画了圈或划去为止

```
public int countPrimes(int n){
	int res = 0; //保存结果计数
	boolean[] nums = new boolean[n];

	for(int i=2; i<nums.length; i++){ //标记全部true
		nums[i] = true;
	}
	//遍历
	for(int i=2; i*i<nums.length; i++){
		if(nums[i]){
			for(int j=i*i; j<nums.length; j+=i){
				nums[j] = false;
			}
		}
	}

	for(boolean b:nums){
		res+=b?1:0;
	}

	return res;
}
```

# 求最优的问题

### 窃贼问题

有一个窃贼带着一个背包去偷东西，房间有3个物品，背包最多4公斤怎么装下金额最多呢？

- 物品1:1公斤1500元
- 物品2：4公斤3000元
- 物品3：3公斤2000元

![](https://ws1.sinaimg.cn/large/005Lno3ugy1g2h1mtcaxyj31bm0h0my0.jpg)

```java
public class Main {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        /**
         * f[i][v] = max{f[i-1][v], f[i-1][v-w[i]+c[i]] }
         * 横坐标为i
         * 纵坐标为v
         * 只考虑放与不放
         *      如果不放第i件物品 -> 将i-1的物品放到背包里 价值[i-1][v]
         *      如果放第i件物品 -> 将把剩余的物品(i--1)放到剩余的背包v-c[i]里
         *
         * */


        int n=3,v=4; //物品的个数与背包的容量
        int[] weight = new int[n+1];
        int[] money = new int[n+1];
        /*输入*/
        for(int i=1; i<=n; i++){
            weight[i] = sc.nextInt();
        }
        for(int i=1; i<=n; i++){
            money[i] = sc.nextInt();
        }

        int[][] dp = new int[n+1][v+1];

        for(int i=1; i<n+1; i++){
            for(int j=1; j<=v; j++){
                if(weight[i] <= j){
                    dp[i][j] = Math.max(dp[i-1][j],dp[i-1][j-weight[i]]+money[i]);
                }else{
                    dp[i][j] =dp[i-1][j];
                }
            }
        }

        /*Print*/
        for(int i=1; i<=n; i++){
            for(int j=1; j<=v; j++){
                System.out.print(dp[i][j] + " ");
            }
            System.out.println();
        }
    }
}
```




# 字符

### 字符移动

函数将字符串中的字符*移到字符串的前面，不能改变字符的先后顺序。比如ab**cd**e*12 转换*****abcde12.

```java
public class Main {
	public static void f(char[] ch) {

		int i;
		int j = ch.length - 1;
		for (i = j; j >= 0; j--) {// i负责找*
			if (ch[i] != '*') {
				i--;
			} else if (ch[j] != '*') {
				ch[i] = ch[j];
				ch[j] = '*';
				i--;
			}
		}

		System.out.println(Arrays.toString(ch));
	}

	public static void main(String[] args) {
		String str = "ab**cd**e*12";
		char ch[] = str.toCharArray();
		f(ch); //[*, *, *, *, *, a, b, c, d, e, 1, 2]
	}
}
```

```java
public class Main {
	static int n = 5;
	static int[] nums = { 1, 3, 5, 7, 9 };
	static int[] arr = new int[n];
	static int[] overf = new int[n];

	public static void dfs(int step) {
		if (step == n) {
			System.out.println(Arrays.toString(arr));
		}

		for (int i = 0; i < n; i++) {
			if (overf[i] == 0) { //是否访问过了
				arr[step] = nums[i];
				overf[i] = 1;

				dfs(step + 1);
				overf[i] = 0;
			}
		}
		return;
	}

	public static void main(String[] args) {
		dfs(0);
	}
}
```

### 解救小哈

```java

import java.util.Scanner;

public class Main {
    static int m;//（m,n）为几行几列
    static int n;
    static int p;//（p，q)为终点
    static int q;
    static int min=9999;
    static int [][]a=new int [51][51];//存放地图
    static int [][]b=new int [51][51];//存放路径

    public static void dfs(int x,int y,int step)
    {
        int [][]next={{0,1},{1,0},{0,-1},{-1,0}};
        if(x==p&y==q)
        {
            System.out.println("step"+step);


            if(step<min)
                min=step;

            return;
        }
        for(int k=0;k<4;k++)
        {
            int tx=x+next[k][0];
            int ty=y+next[k][1];
            if(tx<0||tx>=m||ty<0||ty>=n)
                continue;
            if(a[tx][ty]==0&b[tx][ty]==0)
            {
                b[tx][ty]=1;
                dfs(tx,ty,step+1);
                b[tx][ty]=0;
            }
        }
        return;
    }
    public static void main(String[] args){
        n = 4;
        m = 5;

        a[0][2]=1;
        a[2][2]=1;
        a[3][1]=1;
        a[4][3]=1;

        System.out.println();
        for(int i=0; i<n; i++) {
            for (int j = 0; j < m; j++)
                System.out.print(a[i][j] + "\t");
            System.out.println();
        }

        int startX = 0;
        int startY = 0;
        p = 3;
        q = 2;
        b[startX][startY] = 1; //标记开始
        dfs(startX,startY,0);
        System.out.println(min);
    }
}
```

# 集合

### 求集合的所有子集的算法

对于任意集合A，元素的个数n其子集的个数2^n.在每个子集中，

要么存在，要么不存在

```
(a,b,c)

(1,1,1)->(a,b,c)

(1,1,0)->(a,b  )

(1,0,1)->(a,  c)

(1,0,0)->(a     )

(0,1,1)->(  b,c)

(0,1,0)->(  b   )

(0,0,1)->(     c)

(0,0,0)->@(@表示空集)
```

```java
static void print(int[] arr){
        int len = 1<<arr.length; //2^n
        LinkedList res = new LinkedList();
        for(int i=0; i<len; i++){
            LinkedList list = new LinkedList();
            int index=i;
            for(int j=0; j<arr.length; j++){
                if((index&1) == 1){
                    list.add(arr[j]);
                }
                index>>=1;
            }
            Collections.sort(list);
            res.add(list);
        }
        System.out.println(res);
    }
```
